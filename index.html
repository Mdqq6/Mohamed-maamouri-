Qu’est-ce que le HTML ?

HTML est un langage de balisage utilisé pour structurer le contenu des pages web en utilisant des balises pour définir des éléments comme le texte, les images et les liens hypertexte. Cela permet de créer une structure cohérente et sémantique pour les documents en ligne.

les balises et leur fonction :

la balise <head> :

La balise <head> est une balise HTML utilisée pour inclure des éléments qui fournissent des informations sur le document HTML, mais qui ne sont pas affichés directement à l'utilisateur. Elle contient généralement des métadonnées, des liens vers des feuilles de style, des scripts, et d'autres éléments essentiels pour la structure et le rendu de la page web.

la balise <body> :

La balise <body> est une balise HTML utilisée pour encapsuler le contenu principal d'une page web. Tout ce qui est à l'intérieur de la balise <body> est affiché directement à l'utilisateur, y compris le texte, les images, les liens, les tableaux, etc. Elle constitue la partie visible d'une page web et contient le contenu que les visiteurs voient lorsqu'ils consultent la page dans leur navigateur.

la balise <title> :

La balise <title> en HTML est utilisée pour définir le titre d'une page web, apparaissant dans la barre de titre du navigateur, les onglets et les signets. Elle est cruciale pour le référencement et la compréhension du contenu par les moteurs de recherche.

Les balises <h1> à <h6> sont des éléments HTML utilisés pour définir les niveaux de titres dans une page web. Elles vont de la plus grande (<h1>, titre principal) à la plus petite (<h6>). Ces balises structurent le contenu de la page en indiquant l'importance relative des différentes sections.

la balise <p> (paragraphe) :

La balise <p> en HTML est utilisée pour définir un paragraphe. Elle entoure un bloc de texte pour indiquer qu'il fait partie d'un paragraphe distinct, permettant ainsi une structuration sémantique du contenu sur une page web.

la balise <br> :

La balise <br> en HTML est utilisée pour insérer un saut de ligne. Elle crée un espace vertical entre les éléments qui la contiennent, provoquant le passage à la ligne suivante.

la balise <ul> et <li> :

La balise <ul> en HTML est utilisée pour créer une liste non ordonnée. Elle englobe une série d'éléments <li> (liste items), qui représentent les éléments individuels de la liste.
